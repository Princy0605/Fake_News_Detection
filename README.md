# 📰 Fake News Detection using RNN

## 📌 Project Overview
This project focuses on detecting whether a news article is **Fake** or **Real** using **Deep Learning** techniques.  
The model is trained on textual news data and learns patterns from news content to classify articles accurately.

The project includes:
- Data Cleaning
- Text Preprocessing
- Tokenization
- Deep Learning using RNN
- Model Evaluation

---

# 🎯 Objective
The main objective of this project is to build a deep learning model capable of identifying fake news articles using Natural Language Processing (NLP) and Recurrent Neural Networks (RNN).

---

# 📂 Dataset
The dataset is uploaded for better understanding.

The dataset contains:
- News Title
- News Text
- Labels (Fake / Real)

# 🔄 Project Workflow

## 1. Data Collection
- Imported the WELFake dataset from Kaggle
- Loaded and explored the dataset

## 2. Data Cleaning
- Removed null values
- Removed duplicate records
- Converted text into lowercase
- Cleaned unnecessary symbols and noise

## 3. Text Preprocessing
- Combined title and text columns
- Tokenized textual data
- Converted text into numerical sequences

## 4. Feature Engineering
- Prepared input sequences for deep learning models
- Split dataset into training and testing data

## 5. Model Building
Implemented a Recurrent Neural Network (RNN) model for fake news classification.


## 6. Model Evaluation
The model was evaluated using:
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
